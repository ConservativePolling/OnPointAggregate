## How to Use

Simply open `index.html` in a web browser. Results will automatically update every 30 seconds once polls close at 7:00 PM CT on October 7, 2025.

## Data Source

Powered by [CivicAPI](https://civicapi.org) - Real-time election results API

## Counties Covered

Stewart, Montgomery, Robertson, Houston, Dickson, Cheatham, Hickman, Humphreys, Decatur, Perry, Wayne, Davidson, Williamson, Benton

## Deployment

This site is deployed on Netlify with automatic updates from the main branch.
